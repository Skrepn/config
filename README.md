## 1. Общее описание. 
Данный проект представляет собой инструмент командной строки для преобразования текста из учебного конфигурационного языка в формат XML.

Инструмент читает данные из стандартного ввода, выполняет синтаксический разбор с использованием библиотеки `ply` и выводит результат в стандартный вывод в формате XML.

## 2. Описание всех функций и настроек. 
* t_COMMENT(t): Функция для обработки комментариев. 

* t_CONST_REF(t): Обрабатывает конструкции вида #(NAME). 

* t_NAME(t): Определяет валидные имена переменных (только заглавные латинские буквы).

* t_NUMBER(t): Распознает целые числа (включая отрицательные) и сразу преобразует их из строки в тип int.

* t_newline(t): Отслеживает номера строк во входном файле для корректного отображения ошибок.

* t_error(t): Обрабатывает недопустимые символы, выводя сообщение об ошибке в поток ошибок.

* p_program(p) и p_statements(p): Определяют общую структуру программы как последовательность одного или нескольких выражений.

* p_statement(p): Ключевая функция. Обрабатывает выражение вида ИМЯ := ЗНАЧЕНИЕ.

* p_value_const(p): Реализует логику вычисления констант.

* p_array(p) и p_elements(p): Рекурсивно разбирают массивы вида << ... >>, собирая элементы в список Python.

* p_dictionary(p), p_pairs(p), p_pair(p): Рекурсивно разбирают словари вида [ ... ], собирая пары "ключ-значение" в словарь Python.

* p_error(p): Функция обработки синтаксических ошибок. Сообщает, если структура кода нарушена (например, пропущена запятая или скобка).

* build_xml(doc, parent, data): Рекурсивная функция для преобразования внутренних структур данных Python в XML-дерево.
